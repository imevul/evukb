# Private maintainer docs

Files in this directory (except this README) are **gitignored** and must not be
published.

## SOURCE-MAP.md

Create `docs/private/SOURCE-MAP.md` locally when you need file-level port
checklists or absolute paths to internal sibling repositories used during
clean-room migration.

EvuKB builds and runs without this file. The public migration guide is
[`docs/MIGRATION.md`](../MIGRATION.md).
